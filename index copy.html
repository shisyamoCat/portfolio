About Me

はじめまして。
廣瀬　貴之と申します。

私は2021年10月まで販売業に計18年間従事しておりました。
キャリアやスキルアップに限界を感じ、
以前から興味のあったIT業界へ挑戦すべく職業訓練の門を叩き、
2021年12月に株式会社QLIPの職業訓練コースである
WEBシステム開発実践科を受講しました。

WEBシステム開発実践科では半年かけて

Java Illustrator PhotoShop HTML5 CSS3 JavaScript Servlet＆JSP

といったWEBサイト構築のための知識と技術の学習をしました。

その中でも特に惹かれて学習に熱を入れたのがHTML、CSS、JavaScriptでした。
普段何気なく見ている仕組みも全く分からなかったWEBサイトが、
複数のタグを組み合わせることで作り上げることが出来ることに強い感動と興奮を覚え、
他のカリキュラムと同時並行で自学していた程でした。
結果、半年間の講習でタグの意味すら分からない無学の状態から
卒業制作時にはHTML・CSS・JavaScriptコーディングと調整、不具合修正を一人で担えるようになりました。

とはいえ時間的問題から実装に至らなかった機能や修正できなかった不具合など、
まだまだ未熟な点も多々あり実力不足を痛感している次第ではありますが、
逆境を糧にして学んできた半年間でWEB業界へ踏み出す技術と自信を培いました。

Biography

以前は共に販売業に従事。
計18年間の内、5年を管理職として勤務。
従業員の労務管理や人材育成、商品や店舗などのアセット管理、

2021年12月に新たなことに挑戦するためにITの職業訓練校へ入校。
半年間の講習で主にフロントエンドの知識と技術を研磨。
デザインなどのクリエイティブ系よりもコーディングやプログラミングの方が得意。

負けず嫌いな性格で問題を途中で投げ出すのが嫌い。
集中すれば何時間でも作業が続けられる半面、区切りをつけるのが苦手な側面も。
新しい知識や技術を取り入れることに魅力を感じる。

現在はReactを学習中。
